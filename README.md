# 🚍 APU Smart Transport App

A cross-platform Flutter mobile application designed to improve campus transportation efficiency, including parking management and shuttle booking.

---

## ✨ Features

* 🚗 Parking slot reservation system
* 🚌 Shuttle bus booking with ETA tracking
* 📱 QR-based booking verification
* 👥 Multi-role system (Student, Staff, Admin)
* 📊 Real-time availability updates

---

## 🏗️ Architecture

* Layered Architecture (UI / Service / Model)
* Modular folder structure
* Separation of concerns

---

## 🛠️ Tech Stack

* Flutter (Dart)
* Material UI (Material 3)
* State Management (optional: Provider / Riverpod)
